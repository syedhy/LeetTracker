# Privacy And Ethics

LeetTracker is designed to be a small, respectful macOS utility for tracking public LeetCode progress.

This document is a product/privacy summary, not legal advice.

## What The App Uses

- A LeetCode username entered by the user.
- Public solved-count totals returned for that username.
- Local goal, reminder, and widget settings saved on the Mac.

## What The App Does Not Use

- LeetCode passwords.
- LeetCode session cookies.
- Private submissions.
- Private profile data.
- Automated bulk profile lookup.

## Ethical Use

Use LeetTracker with your own LeetCode username, or with a username you have permission to track. The app is not intended for scraping, bulk monitoring, account enumeration, or competitive intelligence.

## Network Behavior

The app fetches one public profile at a time. Manual refreshes happen only when the user presses Refresh. Background refresh is low-frequency and defaults to every 2 hours.

LeetTracker does not perform bulk profile searches, scrape hidden pages, or bypass authentication.

## Affiliation

LeetTracker is independent and is not affiliated with, endorsed by, or sponsored by LeetCode.
